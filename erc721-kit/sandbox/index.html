Quick Setup Guide

Get your NFT marketplace running in 5 minutes

1

Download the Kit

Download the complete ERC721 Kit package from Gumroad

wget https://your-download-link.zip && unzip erc721-kit.zip
2

Install Dependencies

Install all required dependencies for contracts and frontend

cd erc721-kit && npm install
3

Deploy Contracts

Deploy smart contracts to your preferred network

npx hardhat run scripts/deploy.js --network sepolia
4

Launch Frontend

Start your NFT marketplace application

cd example-app && npm start
